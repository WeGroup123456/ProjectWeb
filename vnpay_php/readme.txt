1. Tạo Cơ sở dữ liệu trong MySQL với tên: vnpay_php_db
2. Import file vnpay_php_db.sql để tạo bảng cho ví dụ demo
3. Cập nhật các tham số cấu hình trong file config.php: kết nối tới máy chủ MySQL và Các tham số VNPAY_TMN_CODE, VNPAY_SECRET_KEY (nếu bạn chưa có thông tin này vui lòng yêu cầu VNPAY gửi lại cho bạn)
4. Copy thư mục code lên máy chủ Apache (WAMP,XAMP,Apache,...) và chạy demo
5. Trong trường hợp bạn chạy bị lỗi, vui lòng liên hệ tới VNPAY để VNPAY hỗ trợ
